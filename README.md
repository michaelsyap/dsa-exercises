# DSA Exercises

A personal collection of Data Structures and Algorithm exercises for continuous learning and skill-building.

## Purpose

This repository documents my hands-on practice with core DSA concepts. Each exercise is worked through as part of ongoing personal training to strengthen problem-solving skills and deepen understanding of fundamental computer science topics.

## Topics

- **Data Structures** — Arrays, Linked Lists, Stacks, Queues, Trees, Graphs, Hash Tables, Heaps
- **Algorithms** — Sorting, Searching, Recursion, Dynamic Programming, Greedy, Backtracking, Graph Traversal
- **Complexity Analysis** — Time and space complexity evaluation for each solution
